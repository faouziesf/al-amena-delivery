✅ Fix: Correction définitive du padding dans le layout client

## Problème résolu
- Les vues client n'avaient pas de padding cohérent
- Contenu collé aux bords de l'écran sur mobile
- Padding dupliqué sur certaines pages

## Solution implémentée
- Ajout du padding directement dans le layout client.blade.php
- Padding automatique pour toutes les vues (mobile: 16px, desktop: 24px)
- Nettoyage de 16 vues pour éviter le double padding
